# 📚 Projetos Escolares - Desenvolvimento de Sistemas

Repositório com projetos e trabalhos desenvolvidos durante o curso de Desenvolvimento de Sistemas.

## 💻 Tecnologias utilizadas

* Python
* Flask
* SQLite
* HTML, CSS e JavaScript

## 🚀 Objetivo

Aplicar na prática conceitos de programação, banco de dados e desenvolvimento web, com foco em backend e criação de APIs.

## 📌 Projetos incluídos

### 🔹 API com Flask

* Criação de rotas (GET, POST)
* Recebimento de dados em JSON
* Integração com banco de dados

### 🔹 Banco de Dados (SQLite)

* Criação de tabelas
* Uso de chave primária e estrangeira (FK)
* Inserção e manipulação de dados

### 🔹 Integração Front-end

* Formulários em HTML
* Envio de dados via JavaScript (fetch)
* Comunicação com API

## 📈 Status

Em constante evolução conforme avanço nos estudos e desenvolvimento de novas habilidades na área de tecnologia.
